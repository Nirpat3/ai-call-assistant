## Contact Management

### Adding Contacts Manually

1. **Navigate to Contacts**
   - Click "Contacts" in the main navigation
   - Click "Add Contact" button

2. **Enter Contact Information**
   - First Name (required)
   - Last Name (required)
   - Phone Number (required, format: +1234567890)
   - Email (optional)
   - Company (optional)
   - Job Title (optional)

3. **Set Contact Preferences**
   - Mark as VIP for priority handling
   - Add custom routing rules
   - Set preferred contact methods

### Mobile Contact Sync

**For iPhone Users:**
1. Install the AI Call Assistant app from the App Store
2. Grant contact access permissions
3. Tap "Sync Contacts" in the app
4. Your iPhone contacts will be imported automatically

**For Android Users:**
1. Install the AI Call Assistant app from Google Play
2. Allow contact permissions
3. Use "Sync Contacts" feature
4. All contacts will be transferred securely

**Web Browser Sync:**
1. Export contacts from your phone as CSV or VCF file
2. In the web dashboard, go to Contacts
3. Click "Import Contacts"
4. Upload your file and map the fields
5. Review and confirm the import

## Call Routing Setup

### Basic Call Routing

**Default Behavior:**
- All calls are answered by AI assistant
- AI attempts to understand caller intent
- Calls are routed based on keywords and context
- Voicemail option always available

### Custom Routing Rules

1. **Create New Route**
   - Go to "Call Routes" section
   - Click "Add Route"
   - Enter route name and description

2. **Configure Route Conditions**
   - **Keywords**: Words that trigger this route
   - **Time Conditions**: Business hours, weekends, holidays
   - **Caller Conditions**: Specific contacts or phone number patterns

3. **Set Route Actions**
   - **Forward to Number**: Transfer call to specific phone
   - **AI Handling**: Let AI manage the conversation
   - **Voicemail**: Send directly to voicemail
   - **Block**: Automatically reject calls

### Contact-Specific Routing

1. **Open Contact Details**
   - Find contact in your contact list
   - Click contact name to open details

2. **Add Routing Rule**
   - Click "Add Route" in contact details
   - Choose action (Forward, AI, Voicemail, Block)
   - Set priority level (1-10)
   - Enable/disable based on business hours

3. **Personalized Greetings**
   - Record custom greeting for specific contacts
   - Use contact's name in AI responses
   - Set different tone (formal, casual, friendly)
